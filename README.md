# 使用说明
## 天气插件
页面天气使用的是和风天气（UI相对好看）  
天气插件在index.html的第24行，使用前请替换成您自己的地址（否则会挤占我的QPS，免费版只支持10个/分钟）  
您可自行去官网申请，或者替换其他天气的脚本。
## 图像资源
相关图像都在image文件夹内，设置均在index.html，CSS内没有指定图像调用，对于小白尝试来说非常合适，相信稍有HTML基础的人都能看得懂。
